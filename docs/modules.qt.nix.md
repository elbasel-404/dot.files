# qt.nix

Configures Qt framework options for theming and platform integration.

## Options Explained

- `qt.style = "kvantum";`
  - (Commented) Sets the Qt style/theme to Kvantum, a popular theme engine for Qt applications.
- `qt.enable = false;`
  - (Commented) Disables Qt integration.
- `qt.platformTheme = "qt5ct";`
  - (Commented) Sets the Qt platform theme to qt5ct, allowing user customization of Qt application appearance.
